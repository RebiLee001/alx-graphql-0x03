# Rick and Morty GraphQL App

This project is part of the **ALX GraphQL module (0x02)**.  
It demonstrates how to set up a **Next.js** application using **TypeScript**, **Tailwind CSS**, **Apollo Client**, and **GraphQL** to fetch data from the Rick and Morty API.
